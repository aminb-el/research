# Collatz Computations

This folder documents older computational experiments related to Collatz-style iterations and generalized Collatz testing.

The main public code archive is here:

```txt id="hryhfy"
https://github.com/Whoknowsme0nobody/Collatz
```

That repository contains browser-based HTML tools for testing generalized Collatz-style rules, convergence rates, divisor parameters, and adaptive switching behavior.

These materials are included only as computational exploration and programming history. They are not presented as a proof of the Collatz conjecture or as a current mathematical claim.

## Scope

This project should be read as:

```txt id="hhyhjy"
computational exploration
old browser-based testing tools
number theory programming practice
abandoned/superseded experimentation
```

It should not be read as:

```txt id="tas2yr"
a proof of Collatz
a current research paper
a formal mathematical result
```
