## AI project

Contains my independent empirical study on instrumental convergence and self-preservation behaviors in large language models.

Main file:

```txt
instrumental_convergence_revised.pdf
```

This project studies model behavior across 2025–2026 using adversarial conversational pressure, agentic scenarios, scoring rubrics, and AI-to-AI testing.

Some sensitive or potentially misuse-enabling details are not included publicly.

## framework

Contains my main truth-probability and importance-scoring work.

This includes:

```txt
truth_importance_framework_relevance_gated.pdf
framework_empirical_study_paper_v3_5.pdf
relevance_gated_empirical_study_v1_methodology_paper_v3_5.pdf
relevance_gated_empirical_study_v1_analysis_summary_v3_5.pdf
```

It also includes supporting archives such as analysis bundles, scorer packets, input archives, output archives, old cases, and framework files.

The central framework evaluates statements through three outputs:

```txt
T_abs(s)      evidence-weighted truth-probability
T_perc(s,r)   perceived truth-probability
I(s,r)        receiver-dependent importance
```

The empirical study evaluates the framework across HumAID, CREDBANK/CredEvent, and LIAR/PolitiFact using blind external variable assignment and owner-side formula computation.

## goldbach

Contains my number theory work on Goldbach representations.

Main files:

```txt
corrected_goldbach_paper.pdf
goldbach_paper_fr.pdf
```

The main paper develops a Goldbach-shift construction for propagating Goldbach representations, proving generated-family lower bounds under known short-interval prime results and conditional prime-gap assumptions, while separating deterministic claims from heuristic prime-selector behavior.
